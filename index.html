Logic 4 ⚡ — Hybrid Generator (Multi-Table + Full Aux Data)

                

                    Extracts all 5 ACCDB tables simultaneously. Builds real BEND, RIGID, RESTRANT, SIF&TEES records
                    from confirmed column names. Exports a Universal CSV (~130 columns) that can
                    re-generate the .CII without the original ACCDB.
                    Core: Logic4_pipeline.build_hybrid_cii().
                

            


            
            

                
                
                
                
            


            
            

                
Output / Processing — Convert to .CII

                

                    

                        
                        
                        
ACCDB/Multiple CSVs: all 5 INPUT_* tables are extracted automatically. Single Universal CSV: re-generates CII with no ACCDB required.

                    

                    

                        
                    

                    

                        
                        
                        
If omitted, a standalone CII skeleton is generated.

                    

                    

                        
                        
                    

                

            


            
            

                
Universal CSV — ~130 Columns (All ACCDB Tables)

                

                    Exports an element-centric CSV with all pipe, thermal, material, and aux data columns
                    from all 5 ACCDB tables. BND_, RST_, RGD_,
                    and SIF_ prefixed columns contain joined aux-table data per element.
                

                

                    This CSV can be edited and re-uploaded to Logic 4 to regenerate the .CII without the original ACCDB.
